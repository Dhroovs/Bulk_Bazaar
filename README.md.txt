# ✨ Overview

Bulk Bazaar is a modern ecommerce platform developed using Laravel and Tailwind CSS.

The project includes:
- customer ecommerce experience
- premium admin dashboard
- product management
- category management
- shopping cart system
- checkout workflow
- order management
- inventory handling
- responsive modern UI

The application was designed with a premium dark-theme ecommerce interface inspired by enterprise ecommerce platforms.

# ⚡ Installation

```bash
git clone https://github.com/Dhroovs/Bulk_Bazaar.git
```

```bash
cd Bulk_Bazaar
```

```bash
composer install
```

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan migrate
```

```bash
php artisan serve
```
